# Heatic Debate App

**Heatic Debate App** is a modern discussion platform designed to help users engage in structured debates, express opinions, and explore diverse perspectives in a respectful and interactive environment.

This Flutter-based version represents a **major upgrade** from the original Java implementation, delivering improved performance, enhanced UI/UX, and a scalable architecture for future features.

---

## 🚀 About the Project

The original **Heatic Debate App** was successfully published on the Google Play Store using **Java (Android native)**.
To support faster development, cross-platform compatibility, and a better user experience, the application is now being **rebuilt from scratch using Flutter**.

This new version introduces:

* ⚡ Faster performance
* 🎨 Modern and responsive UI
* 📱 Cross-platform capability (Android + future iOS support)
* 🔐 Improved authentication system
* ☁️ Firebase integration
* 🧩 Scalable and maintainable architecture
* 🚀 Future-ready feature expansion

---

## ✨ Key Features (Planned & Implemented)
